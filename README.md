## Projects

### [SHA — Stylus Hardware Anchor](https://github.com/arhantbarmate/stylus-hardware-anchor)
Reusable hardware identity verification primitive for Arbitrum Stylus. Binds ESP32-S3 eFuse silicon identifiers to on-chain execution — enabling deterministic device identity, replay-safe receipts, and firmware-governed access control without centralized intermediaries.

- **Prototype:** Deployed on Arbitrum Sepolia
- **Gas (amortized):** 12.5k gas/receipt at N=50 batch
- **Test coverage:** 10,000+ vectors, 100% Keccak-256 parity  
- **Grant:** $25,000 USD application submitted to Stylus Sprint (pending)
- **License:** MIT

### [Nexus Protocol](https://github.com/arhantbarmate/nexus-core)
Universal zero-trust execution layer for DePIN hardware. Moves verification to the edge — validating physical sensor integrity and hardware context before any protocol-level execution occurs.

- **Phase 1.4.0:** Cloudflare Zero Trust ingress, fail-closed Sentry, SQLite WAL vault
- **Stress tested:** 1M transactions, 0% corruption, 50–60 TPS baseline on edge hardware
- **Economic invariant:** 60/30/10 split enforced at Brain level — immutable
- **License:** Apache 2.0

### [OAP — Integrity Governance Engine](https://orthonode.xyz/oap.html)
Deterministic motion validation and lifecycle trust governance for mobility, robotics, and DePIN systems. OAP emits verifiable integrity state; partners apply economic logic.

- **Design:** Physics envelope enforcement, non-linear trust scoring, identity clone containment
- **Status:** Architecture and structural model published, product in development
